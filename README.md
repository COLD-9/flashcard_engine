🧠 AI-Powered Flashcard Engine (智能外刊阅读与记忆引擎)
📌 项目简介
这不仅仅是一个背单词软件，而是一个专为备考（考研/四六级）和高阶英语学习者打造的**“阅读辅助 + 知识提炼 + 科学复习”闭环系统**。
它能自动从用户输入的碎片化英文文章中，精准提取出符合考试大纲的核心难词，利用大模型（LLM）进行语境级翻译与形近词辨析，并基于艾宾浩斯遗忘曲线生成每日复习计划。

🎯 核心解决痛点
“收藏从未看”： 解决碎片化长文阅读效率低下的问题，一键提取核心学习价值。

“脱离语境背词”： 解决传统词典生搬硬套的问题，实现基于当前阅读上下文的精准翻译。

“易混淆词陷阱”： 针对考研/四六级痛点，自动梳理形近词和同义词，构建词汇网络。

🚀 核心功能模块与需求细化 (V1.0 MVP 版本)
模块一：Data Ingestion (智能数据摄入)
🌐 网页正文提取： 用户输入一个纯英文外刊网页链接（如 The Economist, Medium），系统能自动剥离广告和无用标签，提取干净的正文内容。

📸 图像文字识别 (进阶需求)： 用户上传包含英文片段的截图，系统通过本地 OCR 引擎提取文本。

✂️ 原句快照保留 (体验优化)： 在提取生词的同时，必须保留该生词在文章中出现的“原始句子”，供后续复习时唤醒阅读记忆。

模块二：AI Processing (AI 大脑深度加工)
🧹 基础文本清洗： 使用 NLP 工具进行分词、词形还原（如 running -> run），并剔除介词、代词等无意义的停用词 (Stop words)。

📚 考试大纲词库过滤 (硬核功能)： * 将清洗后的文章词汇，与本地内置的 SQLite 考试词库（考研 5500 词 / 四六级词库）进行交集匹配。

仅将命中大纲且用户尚未掌握的词汇送入下一步，避免在简单词（如 apple, the）上浪费 AI 算力和时间。

🤖 大模型 (LLM) 结构化解析： 调用 DeepSeek 等大模型 API，基于“单词 + 语境原句”，生成严格的 JSON 结构数据，包含：

word & phonetic（单词与音标）

translation（结合当前语境的精准中文释义）

example（进阶英文例句）

synonyms（同义词拓展）

confusing_words（形近词/易混淆词避坑指南）

模块三：Memory Scheduling (科学记忆调度)
💾 本地化持久存储： 使用 SQLite + SQLAlchemy 构建轻量级关系型数据库，存储词汇本体数据、复习记录和文章来源。

📈 艾宾浩斯调度算法：

每个单词拥有独立的状态标签（如：新词、复习中、已掌握）。

根据用户复习时的反馈（“认识”、“模糊”、“忘记”），动态计算并更新该单词的 next_review_date（下次复习时间戳）。

📤 数据导出 (体验优化)： 支持将生成的闪卡数据一键导出为 CSV 格式，方便高阶用户导入至 Anki 使用。

模块四：Interaction & UI (交互与终端展示)
💻 极客命令行 (CLI) 交互：

python main.py add <URL/Text>：解析新文章并生成闪卡。

python main.py review：进入沉浸式终端复习模式。

python main.py stat：在终端打印当前词汇量、今日待复习数量等统计信息。

📊 数据可视化 (进阶需求)： 后续暴露 FastAPI 接口，结合前端 D3.js，绘制用户的“每日记忆遗忘曲线”和“大纲词汇覆盖率图表”。

🛠️ 技术栈选型
编程语言： Python 3.10+

核心依赖：

requests / httpx (网络请求与 API 调用)

newspaper3k (网页正文提取)

nltk / spaCy (本地 NLP 基础处理)

SQLAlchemy (ORM 数据库操作)

python-dotenv (环境变量与 API Key 管理)

数据库： SQLite (本地轻量级)

AI 引擎： 兼容 OpenAI 接口格式的大语言模型 (如 DeepSeek-Chat)
